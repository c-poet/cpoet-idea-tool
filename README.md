**CPoet Tools** 是一款 IntelliJ IDEA 插件，集成了常用的开发工具，例如生成增量补丁、Base64 编码/解码、Hash 值计算等。

## 主要功能 🌟

1. **增量补丁生成 📦**
   - 选择文件快速生成 ZIP 格式的增量补丁包 ⚡
   - 自动生成路径说明文件 📝
   - 支持 MapStruct ♻️

2. **文本与编码工具 🔠**
   - 驼峰式和下划线式转换 🔄
   - Base64 编码/解码 🔏
   - 计算 MD5/SHA1 哈希值 🔢

3. **数据库查询辅助 🗄️**
   - 将多个数据库值拼接为 IN 条件 🔗
   - 根据列类型自动判断是否添加单引号 🤖
